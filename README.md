# PYClassicChat
A Minecraft Classic chat client written in Python

# 📌 Features
- Connecting to classic servers ✅
- Support for Minecraft color codes using ANSI ✅
- Ways to teleport and place blocks ❎
- CPE ❎ (coming soon)
